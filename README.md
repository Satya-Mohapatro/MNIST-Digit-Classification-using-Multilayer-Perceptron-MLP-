#  MNIST Digit Classification using Multilayer Perceptron (MLP)

This project implements an end-to-end deep learning pipeline to classify handwritten digits (0–9) from the MNIST dataset using a Multilayer Perceptron (ANN).  
It includes preprocessing, model training, evaluation, visualizations.

---

##  Project Highlights

- ✔️ MNIST Dataset (60k train + 10k test samples)
- ✔️ Preprocessing: normalization + flattening
- ✔️ MLP Model with BatchNorm + Dropout
- ✔️ Training with callbacks (EarlyStopping, Checkpoint, LR Scheduler)
- ✔️ Visualizations:  
  - Loss & Accuracy curves  
  - Confusion Matrix  
  - Saliency Maps  
  - Sample Predictions  
- ✔️ Deployment ready: Flask API + TFLite conversion
- ✔️ Fully modular code structure
